# 智能仓库管理系统 企业版 V2

## 企业版功能

### 已支持

- 用户登录
- 权限管理
- 库存管理
- 库存预警
- 手机端同步
- 出入库记录
- 操作日志
- 美观UI
- Windows11支持

# 安装步骤（电脑新手版）

## 第一步：安装 Node.js

官网：

https://nodejs.org

下载：

LTS版本

安装：

一路 Next 即可。

---

## 第二步：解压项目

建议放：

D:\warehouse-enterprise

---

## 第三步：启动

双击：

start.bat

首次会自动安装依赖。
